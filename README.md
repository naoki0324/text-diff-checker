# テキスト差分チェッカー (Text Diff Checker)

モダンで高速なテキスト比較ツールです。2つのテキスト間の差異を検出し、行単位で分かりやすくハイライト表示します。

![Text Diff Checker Demo](./public/screenshots/demo.png)

## 🌟 特徴

- **1画面完結型レイアウト**: スクロール不要で、入力と比較結果を同時に確認できます。
- **リアルタイム比較**: 差分計算をローカルで行うため、非常に高速です。
- **ファイルアップロード対応**: テキストファイル（.txt, .md等）を直接読み込んで比較できます。
- **レスポンシブデザイン**: デスクトップからモバイルまで最適な表示を提供します。
- **AI 翻訳機能**: Gemini API を使用して、入力テキストや比較結果を「和訳」「英訳」できます。比較結果の翻訳は、**差分のハイライト状態を維持**したまま行われます。
- **Vite + React + Tailwind CSS**: 最新の技術スタックによる軽量・高速な動作。

## 🚀 セットアップと実行

### システム要件

- Node.js (v18以上推奨)
- npm

### インストール

プロジェクトディレクトリで以下のコマンドを実行します。

```bash
npm install
```

### 環境設定

`.env.local` ファイルを作成し、Gemini API キーを設定します（翻訳機能を使用する場合に必要です）。

```text
VITE_GEMINI_API_KEY=あなたのAPIキー
```

### 開発サーバーの起動

```bash
npm run dev
```

起動後、 [http://localhost:3000](http://localhost:3000) にアクセスしてください。

## 📁 プロジェクト構造

- `src/`: ソースコード
  - `App.tsx`: メインのロジックとレイアウト。
  - `index.tsx`: エントリーポイント。
  - `index.css`: グローバルスタイル。
  - `components/`: UIコンポーネント。
  - `lib/`: 共通ロジック、API clients。
  - `types/`: 型定義。
- `public/`: 静的資産（ファビコン、スクリーンショット）。
- `index.html`: エントリーポイント（HTML）。
- `vite.config.ts`: Vite の設定。
- `tsconfig.json`: TypeScript の設定。

## 🛠 使用技術

- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Vite 6
- **Algorithm**: 行ベースの動的計画法 (LCS) による差分検出
